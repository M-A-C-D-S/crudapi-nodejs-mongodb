//Configuração do servidor
1. Utilizar npm run server para iniciar o projeto
2. Inserir dados no .env como conexão de banco e porta que o servidor rodará
3. Importe o arquivo POSTMAN.json no Postman para realizar testes com a API (OPCIONAL)

//Configuração do Postman
1. Edite a coleção, em Variables 
2. Na variável URL, insira a porta onde seu servidor está rodando em http://localhost:PORTA no Initial Value e Current Value. Exemplo: http://localhost:80
3. Salve a alteração com CTRL + S
